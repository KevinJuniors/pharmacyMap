Info

국내 지리정보가 정확히 반영 되어있는 카카오 맵 기반으로 만들어진 약국 웹 지도 서비스
Pharmacy Map 입니다.


항상 사용자 편의를 최우선으로 고려하여 깔끔하고 사용하기 편리한 지도 웹 서비스가
되도록 하겠습니다.

Maps

  • 약국
지도 중심부 기준 주소정보(지번주소 기준)

Skills

Pharmacy Map 제작 과정에서 사용된 스킬들을 소개하는 페이지입니다.

HTML 5

HTML5 스킬을 활용하여 Pharmacy Map 웹 페이지의 기본 레이아웃 구조를 구현하는데 사용 되었습니다.

CSS 3

HTML5 스킬을 활용하여 기본 레이아웃 기틀을 잡은 것에서 좀더 정밀화된 작업
즉, position, padding, margin, float, background, font, overflow 속성 등을 활용하여
시각적으로 보기 좋고 깔끔한 웹 페이지 연출을 위해 활용 되었습니다.

JavaScript

웹 페이지 상에서 보여지는 카카오 맵 기반 API에 대한 전반적인 기능 설정을 하는데 활용된 웹 기술 입니다.

jQuery

웹 페이지 내 스크롤 이벤트를 제어하는데 활용된 JavaScript의 대표적 라이브러리 jQuery 입니다.

Bootstrap

Pharmacy Map 웹 페이지의 부가적인 레이아웃 구성 할 때 주로 활용 되었으며, 해당 웹 페이지는
Bootstrap, jQuery 기반의 오픈 템플릿을 "약국 웹 지도"의 정체성을 지닌 Pharmacy Map 웹 서비스의 취지에 맞춰
제작 하였습니다.
